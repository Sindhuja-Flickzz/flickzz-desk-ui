# Kanban Board - Quick Start Guide

## Accessing the Kanban Board

### Step 1: Navigate to Project Builder
- Click on **Project Builder** from the main menu
- You'll see two tabs: **Create** and **List**

### Step 2: View Project List
- Click on the **List** tab
- All your projects will be displayed in a table

### Step 3: Open Kanban Board
- Find your project in the list
- Click the **Kanban** button (dashboard icon) in the Actions column
- The Kanban board will load with your project's tasks

## Working with Tasks

### Moving Tasks Between Columns

1. **Find the task** you want to move
2. **Click and hold** on the task card
3. **Drag it** to the destination column
4. **Release** the mouse button
5. The task status updates automatically

### Reordering Tasks Within a Column

1. **Click and drag** a task card
2. **Move it up or down** within the same column
3. **Release** to save the new order

### Viewing Task Details

Hover over any task card to see additional information:
- Complete task title
- Description
- Assigned leads' avatars
- Due date
- Work days estimate
- Story points
- Progress percentage

## Filtering & Searching

### Quick Search

1. Type in the **search box** at the top
2. Results filter by title, code, or description in real-time
3. Clear to see all tasks again

### Filter by Epic

1. Click the **Epic** dropdown
2. Select an epic name
3. Board shows only tasks from that epic

### Filter by Lead/Assignee

1. Click the **Lead** dropdown
2. Select a person/company
3. Board shows only their assigned tasks

### Filter by Status

1. Click the **Status** dropdown
2. Select a column/status type
3. Board shows only that column (useful on mobile)

### Show Only Overdue Tasks

1. Check the **Show Overdue** checkbox
2. Only tasks past their due date will display
3. These appear in non-completed columns

## Column Management

### Collapse/Expand Columns

- Click the **↑ or ↓** icon in column header
- Collapsed columns show task count
- Expands to show full task cards

### Task Count

- See total tasks in each column on the header
- Count updates as you move tasks
- Helps track workflow progress

## Tips & Tricks

### Working on Desktop
- Drag across multiple columns smoothly
- Use filters for focused view
- Collapse columns to see more at once

### Working on Tablet
- Horizontal scroll to see more columns
- Touch and drag to move tasks
- Reduced spacing for mobile viewport

### Working on Mobile
- Swipe between columns
- Tap task to see more details
- Use Status filter to work on one column at a time

### Productivity Tips
- Use Search to find urgent tasks
- Filter by Lead to see your team's work
- Show Overdue to catch deadline risks
- Collapse completed columns to reduce clutter

## Common Tasks

### Find a Specific Task
1. Type task name/code in search box
2. Results show matching tasks
3. Click to see in context

### Move All Your Tasks to Done
1. Filter by Lead (yourself)
2. Move each task to Done column
3. Clear filters to see impact

### Check Team Progress
1. Clear all filters (see all tasks)
2. Check task distribution across columns
3. Identify bottlenecks

### Find Overdue Work
1. Check "Show Overdue"
2. These are priority tasks
3. Move them forward ASAP

## Troubleshooting

### Tasks Not Moving?
- Ensure you have drag-drop enabled
- Try refreshing the page
- Check browser console for errors

### Filters Not Working?
- Clear search box first
- Try one filter at a time
- Refresh page if stuck

### Can't See All Columns?
- Collapse some columns to see more
- Use horizontal scroll on mobile
- Try Status filter to focus on one

### Performance Slow?
- Clear search/filters
- Collapse unnecessary columns
- Refresh the page
- Try from different browser
